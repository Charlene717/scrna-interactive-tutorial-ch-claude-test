STEP 8 / 9

細胞註釋 (Annotation)

將數學上的 Cluster 轉譯為有生物學意義的細胞類型名稱。

核心概念

從 Cluster 到細胞類型

聚類不會告訴你細胞類型。註釋的任務是結合標誌基因在特定細胞類型中特異性高表達的基因，如 CD3D→T 細胞與領域知識來標記。

DEA 找 Markers

比較目標 cluster 與其餘細胞，找到表達顯著上升的基因。

自動註釋

SingleR、Azimuth、CellTypist 與參考數據集比對。

專家驗證（黃金標準）

根據文獻 marker gene 列表逐一檢查，不可省略。

互動模擬

Dot Plot 互動探索器

選擇一個 cluster，觀察各 marker 基因的表達量。高表達的基因組合指向特定細胞類型。

長條高度 = 平均表達量 ｜ 根據 marker 基因的表達模式判斷細胞類型

程式碼

實作範例

markers <- FindAllMarkers(pbmc, only.pos = TRUE, min.pct = 0.25)
DotPlot(pbmc, features = c("CD3D","MS4A1","CD14","GNLY")) + RotatedAxis()
new.ids <- c("CD4 T", "CD14 Mono", "B cell", ...)
pbmc <- RenameIdents(pbmc, new.ids)
sc.tl.rank_genes_groups(adata, groupby="leiden", method="wilcoxon")
sc.pl.dotplot(adata, var_names=["CD3D","MS4A1","CD14"], groupby="leiden")
adata.obs["cell_type"] = adata.obs["leiden"].map({"0":"CD4 T","1":"Mono",...})
