STEP 8 / 9

細胞註釋 (Annotation)

將數學上的 Cluster 0, 1, 2… 轉譯為有生物學意義的細胞類型名稱。

核心概念

從 Cluster 到細胞類型

聚類只是數學分群，不會告訴你每個 cluster 是什麼細胞。註釋的任務就是結合已知的標誌基因 (Marker Genes)在特定細胞類型中特異性高表達的基因，如 CD3D 代表 T 細胞、MS4A1 代表 B 細胞與領域知識，為每個 cluster 貼上生物學標籤。

三種策略

註釋方法

差異表達分析 (DEA) 找 Markers

比較目標 cluster 與其餘所有細胞，找出表達量顯著上升的基因。這些基因就是該 cluster 的「候選標誌」。

自動註釋工具

使用 SingleR（R）、Azimuth（Seurat 生態系）或 CellTypist（Python）等演算法，將你的數據與高品質參考數據集進行比對，自動推斷細胞類型。

專家手動驗證（黃金標準）

由領域專家根據文獻中的已知 marker gene 列表，逐一檢查自動註釋的結果。這一步不可省略，因為自動工具可能受參考數據集的品質與完整度影響。

常用 Markers

PBMC 常見細胞類型標誌基因

細胞類型代表 Marker Genes
CD4+ T 細胞CD3D, CD3E, IL7R
CD8+ T 細胞CD3D, CD8A, CD8B
B 細胞MS4A1 (CD20), CD79A
NK 細胞GNLY, NKG7, KLRD1
單核球 (Monocytes)CD14, LYZ, S100A8
樹突細胞 (DC)FCER1A, CST3

上表僅適用於人類 PBMC。不同物種、不同組織的 marker genes 完全不同，務必查閱相關文獻。
程式碼

實作範例

# 找每個 cluster 的 marker genes
markers <- FindAllMarkers(pbmc, only.pos = TRUE, min.pct = 0.25, logfc.threshold = 0.25)

# 查看 top markers
markers %>% group_by(cluster) %>% slice_max(avg_log2FC, n = 5)

# 用已知 markers 視覺化驗證
FeaturePlot(pbmc, features = c("CD3D", "MS4A1", "CD14", "GNLY"))
DotPlot(pbmc, features = c("CD3D", "MS4A1", "CD14")) + RotatedAxis()

# 手動指定細胞類型
new.ids <- c("CD4 T", "CD14 Mono", "B cell", "CD8 T", ...)
names(new.ids) <- levels(pbmc)
pbmc <- RenameIdents(pbmc, new.ids)
# 找 marker genes
sc.tl.rank_genes_groups(adata, groupby="leiden", method="wilcoxon")
sc.pl.rank_genes_groups(adata, n_genes=10)

# 視覺化驗證
sc.pl.umap(adata, color=["CD3D", "MS4A1", "CD14"])
sc.pl.dotplot(adata, var_names=["CD3D", "MS4A1", "CD14"], groupby="leiden")

# 手動指定
cluster_to_celltype = {"0": "CD4 T", "1": "CD14 Mono", ...}
adata.obs["cell_type"] = adata.obs["leiden"].map(cluster_to_celltype)
決策引導

該用手動還是自動註釋？

🌳 決策流程

Q1:
你的組織類型有良好的公開參考數據集嗎？ → 是 → 使用 Azimuth / SingleR 做初步自動註釋，再手動驗證。
Q2:
沒有參考數據集？ → 使用 DEA 找 markers + 文獻比對 做純手動註釋。
Q3:
某些 cluster 的 markers 很混亂、不典型？ → 考慮是否為 doublets、過度聚類、或真正的新亞群——可能需要回頭調整 QC 或 resolution。
