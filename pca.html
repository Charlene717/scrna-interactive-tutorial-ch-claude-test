主成分分析 (PCA)

將數千維壓縮為數十個主成分，保留訊號、過濾噪音。

核心概念

PCA 的角色

即使只用 2000 HVGs，仍是 2000 維——PCA 找出方差最大的方向（主成分），用前 N 個就能捕捉絕大部分訊號。

🗜️

降維

~2000 維 → 10–50 維

🧹

去噪

丟棄低方差主成分

⚡

加速

後續計算大幅提升

互動模擬

Elbow Plot：選擇多少個主成分？

拖動滑桿設定你的選擇，觀察捕捉的方差比例變化。

程式碼

實作範例

pbmc <- RunPCA(pbmc, features = VariableFeatures(pbmc))
ElbowPlot(pbmc, ndims = 50)
# 可選：JackStraw 檢定
pbmc <- JackStraw(pbmc, num.replicate = 100)
sc.tl.pca(adata, n_comps=50)
sc.pl.pca_variance_ratio(adata, n_pcs=50)
