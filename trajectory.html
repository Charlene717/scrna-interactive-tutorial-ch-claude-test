軌跡分析 (Trajectory / Pseudotime)

從靜態快照推估細胞狀態轉變的動態路徑。

核心概念

什麼是 Pseudotime？

scRNA-seq 是一張「快照」。但因為細胞處於不同分化階段，可以利用表達漸變重建擬時序 (Pseudotime)連續數值軸，代表細胞沿生物學過程的進展程度。

🎯

適用情境

分化、免疫活化、疾病進程。

⚠️

前提假設

數據中必須存在不同狀態的細胞。

互動模擬

Pseudotime 與基因表達動態

切換不同的著色模式，觀察 pseudotime 梯度與沿軌跡變化的基因表達。選擇基因觀察其沿 pseudotime 的表達趨勢。

方法比較

主流工具

工具核心邏輯適用
Monocle3學習主圖通用型，支援分支
Slingshot擬合主曲線線性/簡單分支
RNA Velocityspliced/unspliced 比率推斷方向
PAGA圖抽象大規模拓撲

🌳 我該用哪個？

想知道順序？
→ Monocle3 或 Slingshot。
想知道方向？
→ RNA Velocity。
大規模拓撲？
→ PAGA。
程式碼

實作範例

library(monocle3)
cds <- as.cell_data_set(pbmc)
cds <- learn_graph(cds)
cds <- order_cells(cds)
plot_cells(cds, color_cells_by = "pseudotime")
import scvelo as scv
scv.pp.filter_and_normalize(adata, min_shared_counts=20)
scv.pp.moments(adata, n_pcs=30, n_neighbors=30)
scv.tl.velocity(adata)
scv.tl.velocity_graph(adata)
scv.pl.velocity_embedding_stream(adata, basis="umap")
🚨
Pseudotime ≠ 真實時間。不同起始點、不同方法可能給出不同軌跡。需實驗驗證。
