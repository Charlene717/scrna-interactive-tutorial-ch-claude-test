STEP 7 / 9

UMAP 視覺化

將高維數據投影到 2D 平面，直觀觀察細胞群間的相對關係——但千萬別過度解讀。

核心概念

UMAP 是什麼？

UMAPUniform Manifold Approximation and Projection：一種非線性降維方法，將高維拓撲結構近似投影到低維空間將 PCA 空間中的細胞關係投影到 2D 平面上。它不是分析的核心步驟（聚類是在 PCA 空間中完成的），而是一個視覺化工具，幫助我們「看到」分群結果。

🚨
常見誤讀警告：UMAP 圖上群體間的距離不代表生物學相似度；群體的大小也不代表細胞數量。這些都是投影的副產品，不應用於生物學推論。
方法比較

UMAP vs t-SNE

UMAP（推薦）t-SNE
速度較快較慢（尤其大數據集）
全局結構較好地保留群體間相對位置主要保留局部結構
可重複性設固定 seed 即可重現結果較不穩定
目前地位scRNA-seq 的標準選擇仍有使用但逐漸被取代
互動模擬

n_neighbors 參數的影響

n_neighbors 控制 UMAP 在保留局部 vs 全局結構之間的平衡。低值強調局部細節，高值產生更連續的投影。

程式碼

實作範例

# 執行 UMAP
pbmc <- RunUMAP(pbmc, dims = 1:15)

# 視覺化（以 cluster 著色）
DimPlot(pbmc, reduction = "umap", label = TRUE)

# 視覺化特定基因的表達
FeaturePlot(pbmc, features = c("CD3D", "MS4A1", "CD14"))
# 執行 UMAP
sc.tl.umap(adata)

# 視覺化
sc.pl.umap(adata, color=["leiden"])

# 特定基因
sc.pl.umap(adata, color=["CD3D", "MS4A1", "CD14"])

A. 這兩種細胞在生物學上非常相似
B. 這兩種細胞共享大量 marker 基因
C. 不一定有生物學意義——UMAP 距離不能直接解讀為相似度
