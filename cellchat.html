ADVANCED

細胞通訊 (Cell-Cell Communication)

透過 ligand-receptor interaction 分析，探索細胞間的訊號傳遞與微環境調控網絡。

核心概念

細胞如何「對話」？

在組織微環境中，細胞並非孤立存在——它們透過分泌配體 (Ligand)由一個細胞分泌的信號分子，如細胞因子、生長因子、趨化因子，與鄰近細胞表面的受體 (Receptor)細胞表面蛋白，能辨識並結合特定的配體，觸發下游訊號傳導結合，觸發訊號傳導。scRNA-seq 能同時測量所有細胞的基因表達，因此可以推斷哪些細胞在「發送」什麼訊號、哪些細胞在「接收」。

方法比較

主流分析工具

工具核心邏輯特色
CellChat (R)基於質量作用定律建模 L-R 交互機率豐富的視覺化（圓形圖、氣泡圖）、支援跨條件比較
CellPhoneDB (Py)基於排列檢定的 L-R 配對篩選支援多亞基複合體、社群活躍
NicheNet (R)從配體到下游基因的調控網路推斷不只看 L-R 配對，還追蹤下游效應
LIANA (R/Py)整合多種方法的統一框架可同時跑多種演算法並取共識

🌳 我該用哪個工具？

Q1:
你需要漂亮的視覺化來呈現結果？ → CellChat，視覺化能力最強。
Q2:
你想知道配體對下游基因的影響？ → NicheNet，能追蹤調控路徑。
Q3:
你想跑多種方法取共識？ → LIANA，統一框架。
程式碼

CellChat 實作範例

library(CellChat)

# 從 Seurat 物件建立 CellChat 物件
cellchat <- createCellChat(pbmc, group.by = "cell_type")

# 載入 ligand-receptor 資料庫
CellChatDB <- CellChatDB.human  # 或 CellChatDB.mouse
cellchat@DB <- CellChatDB

# 計算交互作用
cellchat <- subsetData(cellchat)
cellchat <- identifyOverExpressedGenes(cellchat)
cellchat <- identifyOverExpressedInteractions(cellchat)
cellchat <- computeCommunProb(cellchat)
cellchat <- computeCommunProbPathway(cellchat)
cellchat <- aggregateNet(cellchat)

# 視覺化
netVisual_circle(cellchat@net$count)
netVisual_bubble(cellchat, sources.use = 1, targets.use = c(2,3,4))
from cellphonedb.src.core.methods import cpdb_analysis_method

# 準備輸入檔案
# counts_file: 正規化後的表達矩陣
# meta_file: 細胞類型註釋
cpdb_results = cpdb_analysis_method.call(
    cpdb_file_path="cellphonedb.zip",
    meta_file_path="meta.txt",
    counts_file_path="counts.h5ad",
    counts_data="hgnc_symbol",
    output_path="./output"
)

# 結果包含 means.txt 和 pvalues.txt
注意事項

解讀時的注意事項

⚠️
這些分析都是「推斷」，不是直接證據。scRNA-seq 只能告訴我們配體和受體的 RNA 是否被表達，但 RNA 表達不等於蛋白質表達，更不等於真正的物理交互作用。重要的結果務必用實驗（如共培養、阻斷實驗）驗證。
