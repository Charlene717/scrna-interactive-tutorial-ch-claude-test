STEP 9 / 9

差異表達分析 (Differential Expression)

比較不同群體或條件間的基因表達差異，找出生物學上有意義的基因。

兩種應用

DE 分析的兩大情境

🏷️

找 Marker Genes

比較某個 cluster 與其餘所有細胞，找出該 cluster 的特異性基因。這是細胞註釋的基礎。

⚗️

跨條件比較

比較同一種細胞在不同實驗條件下（如疾病 vs 健康）的表達差異。這是挖掘疾病機制的核心。

方法比較

主要 DE 檢定方法

方法說明優缺點
Wilcoxon非參數檢定，比較兩群的秩次分布✅ 快速穩健 ⚠️ 不考慮零膨脹
MAST專為 scRNA-seq 設計的 hurdle model✅ 處理零膨脹 ⚠️ 較慢
DESeq2 / edgeRBulk RNA-seq 經典方法✅ 成熟穩定 ⚠️ 需做 pseudobulk 聚合
Pseudobulk先將同一樣本的細胞聚合，再用 bulk 方法✅ 正確處理生物重複 ⚠️ 需多個生物重複

🌳 該選哪種方法？

情境1:
找 cluster marker genes（註釋用）→ Wilcoxon 即可，快速且夠用。
情境2:
跨條件比較，有多個生物重複 → Pseudobulk + DESeq2（目前最佳實踐）。
情境3:
跨條件比較，但只有 1 個樣本 → MAST，但要注意統計效力不足。
🚨
常見錯誤：直接用 FindMarkers 做跨條件比較（把每個細胞當作獨立觀察值）。這會嚴重膨脹樣本量，產生大量假陽性。正確做法是先做 pseudobulk 聚合，以生物重複為分析單位。
程式碼

實作範例

# 情境 1：找 Cluster 0 的 markers
cluster0_markers <- FindMarkers(pbmc, ident.1 = 0, only.pos = TRUE)

# 情境 2：跨條件比較（Pseudobulk 做法）
# Step 1: 建立 pseudobulk
pseudo <- AggregateExpression(pbmc,
  group.by = c("cell_type", "sample_id"),
  return.seurat = TRUE
)
# Step 2: 用 DESeq2 進行差異分析
de_results <- FindMarkers(pseudo,
  ident.1 = "disease", ident.2 = "control",
  test.use = "DESeq2"
)

# 火山圖 (Volcano Plot)
EnhancedVolcano(de_results, lab = rownames(de_results),
  x = "avg_log2FC", y = "p_val_adj")
# 情境 1：找各 cluster 的 markers
sc.tl.rank_genes_groups(adata, groupby="leiden", method="wilcoxon")
sc.pl.rank_genes_groups(adata, n_genes=10)

# 情境 2：跨條件比較
# 以特定細胞類型子集進行比較
t_cells = adata[adata.obs["cell_type"] == "CD4 T"]
sc.tl.rank_genes_groups(t_cells, groupby="condition", method="wilcoxon")

# 推薦：使用 decoupler 做 pseudobulk
import decoupler as dc
pdata = dc.get_pseudobulk(adata, sample_col="sample_id",
  groups_col="cell_type", min_cells=10)

A. 計算太慢
B. 無法處理零值
C. 會把同一個體的多個細胞當獨立觀察，嚴重膨脹樣本量導致假陽性
