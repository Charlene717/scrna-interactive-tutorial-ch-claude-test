差異表達分析 (DE)

比較不同群體或條件間的基因差異，找出有意義的基因。

兩種應用

DE 的兩大情境

🏷️

找 Marker Genes

某 cluster vs 其餘 → 細胞註釋基礎。

⚗️

跨條件比較

疾病 vs 健康 → 挖掘疾病機制核心。

互動模擬

火山圖 (Volcano Plot) 模擬器

調整 log2FC 和 p-value 閾值，觀察有多少基因被判定為顯著差異。

方法比較

DE 檢定方法

方法說明適用
Wilcoxon非參數秩次檢定找 markers（快速）
MASTHurdle model，處理零膨脹跨條件，單樣本
Pseudobulk聚合後用 DESeq2跨條件（最佳實踐）
🚨
常見錯誤：直接用 FindMarkers 做跨條件比較會膨脹樣本量產生假陽性。正確做法是 pseudobulk 聚合。
程式碼

實作範例

# 找 markers
cluster0_markers <- FindMarkers(pbmc, ident.1 = 0, only.pos = TRUE)
# Pseudobulk 跨條件比較
pseudo <- AggregateExpression(pbmc, group.by = c("cell_type", "sample_id"), return.seurat = TRUE)
de <- FindMarkers(pseudo, ident.1 = "disease", ident.2 = "control", test.use = "DESeq2")
sc.tl.rank_genes_groups(adata, groupby="leiden", method="wilcoxon")
# Pseudobulk
import decoupler as dc
pdata = dc.get_pseudobulk(adata, sample_col="sample_id", groups_col="cell_type")
