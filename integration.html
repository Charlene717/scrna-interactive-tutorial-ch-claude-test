ADVANCED

數據整合 (Integration)

合併多個樣本或批次的數據，消除技術性 batch effect，保留真實的生物學差異。

核心概念

什麼是批次效應？

當你合併來自不同實驗批次、不同定序日期、不同實驗室的 scRNA-seq 數據時，聚類結果可能主要反映「樣本來源」而非「細胞類型」。這是因為每次實驗都有微小的系統性差異——試劑批號、操作手法、定序平台——這些批次效應 (Batch Effect)與生物學無關的系統性技術差異，導致同一種細胞在不同批次中呈現不同的表達模式會掩蓋真正的生物學訊號。

💡
判斷是否需要 Integration：如果你的 UMAP 上，細胞主要按「樣本」分群而非按「細胞類型」分群，就需要進行 Integration。
方法比較

主流整合方法

方法核心邏輯速度適用場景
Harmony在 PCA 空間中迭代校正⚡ 極快快速整合、大數據集；目前最常用
CCA (Seurat v3)找跨批次共享的相關結構🔵 中等批次間細胞組成相似時
RPCA (Seurat v5)互惠 PCA 投影🔵 中等批次間細胞組成差異較大時
scVI變分自編碼器 (深度學習)🐢 較慢（需 GPU）超大數據集、複雜批次結構
scanorama全景拼接演算法🔵 中等多批次、Python 生態系

🌳 我該用哪種方法？

Q1:
初次嘗試 / 快速探索？ → Harmony，最快、效果穩健。
Q2:
批次間的細胞組成差異大嗎？（如一個有腫瘤、另一個沒有）→ 是 → RPCA，避免強行對齊不存在於另一批次的細胞。
Q3:
數據集超過 50 萬細胞？ → Harmony（最快）或 scVI（準確度可能更高，但需 GPU）。
Q4:
批次結構很複雜（多中心、多時間點、多技術）？ → scVI，深度學習模型更能處理複雜的混雜因素。
程式碼

實作範例

library(harmony)

# 先正常做 NormalizeData → FindVariableFeatures → ScaleData → RunPCA
# 然後在 PCA 空間中執行 Harmony
pbmc <- RunHarmony(pbmc, group.by.vars = "batch")

# 後續步驟使用 Harmony 校正後的 embedding
pbmc <- FindNeighbors(pbmc, reduction = "harmony", dims = 1:20)
pbmc <- FindClusters(pbmc, resolution = 0.5)
pbmc <- RunUMAP(pbmc, reduction = "harmony", dims = 1:20)
# Seurat v5 整合流程
# 資料已包含多個 layers (每個批次一個 layer)
pbmc[["RNA"]] <- split(pbmc[["RNA"]], f = pbmc$batch)

# 選擇整合方法
pbmc <- IntegrateLayers(pbmc,
  method = RPCAIntegration(),  # 或 CCAIntegration(), HarmonyIntegration()
  orig.reduction = "pca",
  new.reduction = "integrated.rpca"
)

# 合併 layers 後繼續下游分析
pbmc[["RNA"]] <- JoinLayers(pbmc[["RNA"]])
import scvi

# 準備數據
scvi.model.SCVI.setup_anndata(adata, batch_key="batch")

# 訓練模型
model = scvi.model.SCVI(adata, n_latent=30)
model.train(max_epochs=100)

# 取得整合後的 latent embedding
adata.obsm["X_scVI"] = model.get_latent_representation()

# 用整合後的 embedding 做聚類和 UMAP
sc.pp.neighbors(adata, use_rep="X_scVI")
sc.tl.leiden(adata)
sc.tl.umap(adata)
驗證

如何評估整合效果？

👁️

視覺檢查

在 UMAP 上分別以「批次」和「細胞類型」著色。好的整合應讓同一細胞類型跨批次混合，但不同細胞類型仍然分開。

📊

定量指標

使用 kBET（批次混合度）和 ASW（silhouette width，生物學保守度）等指標。理想結果是高混合度 + 高保守度。

⚠️
過度整合的風險：如果某種細胞真的只存在於某個條件（如腫瘤細胞只在疾病樣本中），強行整合可能會錯誤地將它與其他細胞合併。務必保留生物學差異。
