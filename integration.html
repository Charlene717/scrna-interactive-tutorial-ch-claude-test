ADVANCED

數據整合 (Integration)

合併多批次數據，消除技術性 batch effect，保留真實生物學差異。

核心概念

什麼是批次效應？

不同實驗批次的系統性差異（試劑、操作、平台）會讓聚類反映「樣本來源」而非「細胞類型」。

💡
判斷標準：UMAP 上細胞主要按「樣本」而非「類型」分群 → 需要 Integration。
互動模擬

整合前後 UMAP 比較

切換觀察整合前（按批次分群）與整合後（按細胞類型分群）的效果。可分別用批次或細胞類型著色。

方法比較

主流方法

方法速度適用
Harmony⚡ 極快通用、大數據集
RPCA🔵 中等細胞組成差異大
scVI🐢 需GPU複雜批次結構

🌳 我該用哪種？

快速探索？
→ Harmony。
組成差異大？
→ RPCA。
>50 萬細胞？
→ Harmony 或 scVI。
程式碼

實作範例

library(harmony)
pbmc <- RunHarmony(pbmc, group.by.vars = "batch")
pbmc <- FindNeighbors(pbmc, reduction = "harmony", dims = 1:20)
pbmc <- RunUMAP(pbmc, reduction = "harmony", dims = 1:20)
import scvi
scvi.model.SCVI.setup_anndata(adata, batch_key="batch")
model = scvi.model.SCVI(adata, n_latent=30)
model.train(max_epochs=100)
adata.obsm["X_scVI"] = model.get_latent_representation()
sc.pp.neighbors(adata, use_rep="X_scVI")
